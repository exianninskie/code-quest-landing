# 🏰 Code Quest Landing Page

An epic portal to a gamified world where logic meets legend. Built with modern web technologies to introduce adventurers to the realm of Code Quest.

## ✨ Features

- **Responsive Design** - Perfect experience on all devices
- **Modern UI** - Glass-morphism design with smooth animations
- **Interactive Elements** - Hover effects and micro-interactions
- **Performance Optimized** - Fast loading and smooth scrolling
- **GitHub Pages Ready** - Easy deployment with GitHub Actions

## 🚀 Quick Start

1. Clone this repository
   ```bash
   git clone https://github.com/exianninskie/code-quest-landing.git
   cd code-quest-landing
   ```

2. Open in your browser
   ```bash
   # Simply open index.html in your preferred browser
   open index.html
   ```

3. Deploy to GitHub Pages
   - Go to repository Settings → Pages
   - Select "Deploy from a branch"
   - Choose main branch and / (root) folder
